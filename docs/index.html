LIMPACAT Tutorial

Your Guide to Getting Started

Introduction

Welcome to the tutorial! This document will guide you through the steps to set up and use our software. Follow each section carefully for best results.

Requirements

Before starting, make sure you have the following installed:

  • Operating System: Windows, macOS, or Linux
  • Required Software: Python, Conda, Docker, and Git

Setup Instructions

  • Step 1: Download the Software

    Pull docker file:

    docker pull yenjungchiu/limpacat

  • Step 2: Analyze Immune Cell Composition from Gene Expression Data

    Navigate to the project directory and use the following command to analyze the immune cell composition from gene expression data:

    python analyze_composition.py --input gene_expression_data.txt

    Example Usage: Use --input to specify your gene expression data file. Additional parameters such as --method and --threshold allow for personalized analysis.

  • Step 3: Perform Survival Analysis Based on Cell Composition

    Run the following command to analyze survival based on cell composition:

    python survival_analysis.py --composition cell_composition_data.txt

    Example Usage: Use --composition to specify the file with cell composition data. You can also customize analysis with parameters like --model and --timeframe to adjust the survival analysis model.

  • Step 4: Prepare Image Data for Training

    Execute the command below to create the necessary image data for training:

    python prepare_image_data.py --input image_folder/ --output training_data/

    Example Usage: Use --input for the folder containing raw images and --output for specifying where the processed data will be saved. Adjust other parameters, such as --resize and --format, for custom preprocessing.

  • Step 5: Train WSI Images for Immune Cell Content Prediction

    Train the model on Whole Slide Images (WSI) to predict high and low immune cell content associated with survival:

    python train_model.py --training_data training_data/ --validation_data validation_data/

    Example Usage: Use --training_data and --validation_data for specifying the directories with training and validation images, respectively. Parameters like --epochs and --batch_size allow fine-tuning of the training process.

Troubleshooting

If you encounter any issues, refer to the GitHub LIMPACAT page for support or submit a new issue.
